# homebrew-tap

Homebrew formulae for [django23](https://github.com/django23)'s tools.

## Install

```sh
brew install django23/tap/<formula>
```

## Available formulae

| Formula | Description |
|---|---|
| [asimov](Formula/asimov.rb) | Exclude development dependencies from Time Machine backups |

## Updating a formula

Formulae are updated manually after an upstream release. See each source repo's `CONTRIBUTING.md` for the release flow.
